# Email Refiner

A Chrome extension that helps you refine and improve emails as you compose them in Gmail.

## What it does

Adds AI-assisted email refinement directly inside Gmail's compose window. Select text or use the popup to get a polished rewrite before sending.

## Tech

Manifest V3 Chrome extension. Content script injected into mail.google.com. Background service worker handles the API calls.

## Install (unpacked)

Clone this repo. Go to chrome://extensions and enable Developer Mode. Click "Load unpacked" and select this folder. Open Gmail and start composing.
